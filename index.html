Overview

Math Exercise is a simple project to create random interactive worksheet of mathematical problems and correct the answer. It is aimed for elementary school level.

Planned feature

  • Support multiple difficulty levels
  • Add rewards picture (medals or something similar)
  • Support more operations
  • Support persistance to save a user records
  • Improve CSS
